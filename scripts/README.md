# Scripts Directory

This directory contains utility scripts for development, maintenance, and data management.

## How to Use
- Activate your virtual environment before running scripts:
  ```sh
  source ../venv/bin/activate
  ```
- Run a script:
  ```sh
  python <script_name>.py [args]
  ```
